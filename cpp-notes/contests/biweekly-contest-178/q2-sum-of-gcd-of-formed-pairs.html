Q2. Sum of GCD of Formed Pairs

📘 題目敘述

給你一個長度為 n 的整數陣列 nums。

請先建立陣列 prefixGcd，對每個 index i：

  • mxi = max(nums[0], nums[1], ..., nums[i])
  • prefixGcd[i] = gcd(nums[i], mxi)

建立完後先排序 prefixGcd，再每次用最小未配對與最大未配對元素配對，累加每組 gcd。若 n 是奇數，排序後中間元素忽略。

條件限制

  • 1 <= n == nums.length <= 10^5
  • 1 <= nums[i] <= 10^9

🧠 解題思路

依題意分三段：先建出 prefixGcd、再排序、最後雙指標由兩端往中間配對並加總每組 gcd。

掃描 nums 時持續維護前綴最大值 mxi。每到 i 就先更新 mxi，再算 pfg[i] = gcd(nums[i], mxi)。

排序後使用 l、r 指向最小與最大未配對值，反覆計算 gcd(pfg[l], pfg[r]) 累加。while (l < r) 會自然略過奇數長度的中間值。

所有變數

  • n：陣列長度
  • pfg：題目中的 prefixGcd 陣列
  • mxi：目前前綴最大值
  • ans：配對 gcd 總和
  • l、r：左右雙指標

🪜 主要流程步驟

1. 先準備 prefixGcd 陣列和前綴最大值

int n = nums.size();
vector<int> pfg(n);
int mxi = INT_MIN;

2. 掃一次 nums，依題意建出 prefixGcd

for (int i = 0; i < n; i++) {
    mxi = max(mxi, nums[i]);
    pfg[i] = gcd(nums[i], mxi);
}

3. 把 prefixGcd 排序

sort(pfg.begin(), pfg.end());

4. 用雙指標從最左和最右開始配對

long long ans = 0;
int l = 0, r = n - 1;
while (l < r) {
    ans += gcd(pfg[l], pfg[r]);
    l++;
    r--;
}

5. 最後回傳答案

return ans;

⏱️ 複雜度

  • 時間複雜度：O(n log n)
  • 空間複雜度：O(n)

💻 程式碼實作

class Solution {
public:
    long long gcdSum(vector<int>& nums) {
        int n = nums.size();
        vector<int> pfg(n);
        int mxi = INT_MIN;
        for (int i = 0; i < n; i++) {
            mxi = max(mxi, nums[i]);
            pfg[i] = gcd(nums[i], mxi);
        }
        sort(pfg.begin(), pfg.end());
        long long ans = 0;
        int l = 0, r = n - 1;
        while (l < r) {
            ans += gcd(pfg[l], pfg[r]);
            l++;
            r--;
        }
        return ans;
    }
};
