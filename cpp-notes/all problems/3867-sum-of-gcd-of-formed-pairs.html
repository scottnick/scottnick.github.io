3867. Sum of GCD of Formed Pairs

📘 題目敘述

建立 prefixGcd 後排序，依序把最小與最大未配對元素配對，回傳所有配對 gcd 的總和。

條件限制

  • 1 <= n == nums.length <= 10^5
  • 1 <= nums[i] <= 10^9

🧠 解題思路

先線性掃描建出 pfg（每格是當前值和前綴最大值的 gcd），再排序後雙指標配對並累加每組 gcd。

所有變數

  • pfg：prefixGcd 陣列
  • mxi：前綴最大值
  • l、r：左右雙指標
  • ans：答案

🪜 主要流程步驟

1. 建立 prefixGcd

for (int i = 0; i < n; i++) {
    mxi = max(mxi, nums[i]);
    pfg[i] = gcd(nums[i], mxi);
}

2. 排序

sort(pfg.begin(), pfg.end());

3. 雙指標配對累加

while (l < r) {
    ans += gcd(pfg[l], pfg[r]);
    l++;
    r--;
}

⏱️ 複雜度

  • 時間複雜度：O(n log n)
  • 空間複雜度：O(n)

💻 程式碼實作

class Solution {
public:
    long long gcdSum(vector<int>& nums) {
        int n = nums.size();
        vector<int> pfg(n);
        int mxi = INT_MIN;
        for (int i = 0; i < n; i++) {
            mxi = max(mxi, nums[i]);
            pfg[i] = gcd(nums[i], mxi);
        }
        sort(pfg.begin(), pfg.end());
        long long ans = 0;
        int l = 0, r = n - 1;
        while (l < r) {
            ans += gcd(pfg[l], pfg[r]);
            l++;
            r--;
        }
        return ans;
    }
};
