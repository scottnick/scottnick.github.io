3866. First Unique Even Element

📘 題目敘述

給你一個整數陣列 nums，請回傳第一個只出現一次的偶數；若不存在回傳 -1。

條件限制

  • 1 <= nums.length <= 100
  • 1 <= nums[i] <= 100

🧠 解題思路

先統計偶數出現次數，再按原陣列順序找第一個出現次數為 1 的值。

所有變數

  • count[101]：記錄偶數出現次數

🪜 主要流程步驟

1. 初始化計數陣列

int count[101] = {0};

2. 統計偶數次數

for (int i : nums) {
    if (i % 2 == 0) {
        count[i]++;
    }
}

3. 依原順序找第一個 unique even

for (int j : nums) {
    if (count[j] == 1) {
        return j;
    }
}

4. 找不到回傳 -1

return -1;

⏱️ 複雜度

  • 時間複雜度：O(n)
  • 空間複雜度：O(1)

💻 程式碼實作

class Solution {
public:
    int firstUniqueEven(vector<int>& nums) {
        int count[101] = {0};
        for (int i : nums) {
            if (i % 2 == 0) {
                count[i]++;
            }
        }
        for (int j : nums) {
            if (count[j] == 1) {
                return j;
            }
        }
        return -1;
    }
};
