3832. Merge Adjacent Equal Elements

📘 題目敘述

給你一個整數陣列 nums。

你必須重複進行以下合併操作，直到再也無法產生變化為止：

如果目前陣列中存在任意一對相鄰且相等的元素，請選擇最左邊的那一對相鄰元素，並把它們替換成一個新元素，新元素的值等於它們的總和。

每次合併後，陣列長度會減少 1。你要在更新後的陣列上持續重複操作，直到無法再合併。

請回傳所有可能的合併完成後的最終陣列。

條件限制

  • 1 <= nums.length <= 10^5
  • 1 <= nums[i] <= 10^5

🧠 解題思路

若每一步都重掃整個陣列找「最左邊相鄰相等」，最壞情況下會很慢，容易 TLE。

我改用堆疊 st 來維護「目前前綴經過完整合併後的結果」。每次讀到新值 now，只需要看 st 末端是否和 now 相等：

  • 不相等：直接把 now push 進去。
  • 相等：依規則合併，pop 末端後令 now *= 2。

合併後的 now 可能再跟新的末端相等，所以用 while 連鎖處理到底，再 push 回去。

整個陣列掃完時，st 就是最終答案。

所有變數

  • st：模擬合併結果的堆疊（vector<long long>）
  • now：目前元素，會在連鎖合併時翻倍

🪜 主要流程步驟

1. 初始化堆疊

建立空的 st。

2. 從左到右掃每個元素

每次先把元素設為 now。

3. while 連鎖合併

只要 st 不空且 st.back() == now，就 pop 並令 now *= 2。

4. 將結果放回堆疊

合併不能再進行時，把 now push 回 st。

5. 回傳最終陣列

掃描結束後直接回傳 st。

💻 程式碼實作

class Solution {
public:
    vector<long long> mergeAdjacent(vector<int>& nums) {
        vector<long long> st;
        for (int i : nums) {
            long long now = i;
            while (!st.empty() && st.back() == now) {
                st.pop_back();
                now *= 2;
            }
            st.push_back(now);
        }
        return st;
    }
};
