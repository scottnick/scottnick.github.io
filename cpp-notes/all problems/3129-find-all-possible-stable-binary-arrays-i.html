3129. Find All Possible Stable Binary Arrays I

練習日期：2026-03-09

難度：Medium

類型：Dynamic Programming、Prefix Sum

📘 題目敘述

給你三個正整數 zero、one 和 limit。

如果一個二元陣列 arr 滿足以下條件，就稱它是 stable：

  • arr 中 0 出現的次數恰好是 zero
  • arr 中 1 出現的次數恰好是 one
  • arr 中每一個長度大於 limit 的子陣列，都必須同時包含 0 和 1

請回傳 stable binary arrays 的總數。由於答案可能非常大，請將結果對 10^9 + 7 取模後回傳。

條件限制

  • 1 <= zero, one, limit <= 200

🧠 解題思路

核心在於：不能只記錄目前用了幾個 0 和幾個 1，還要知道「尾端是什麼數字」以及「尾端連續長度」。

因為題目要求任何長度大於 limit 的子陣列都不能是同一個值，等價於：

  • 不能有連續超過 limit 個 0
  • 不能有連續超過 limit 個 1

因此用四維 DP 最直觀：

dp[z][o][b][l]：已使用 z 個 0、o 個 1，最後數字是 b，尾端連續長度為 l 的合法方案數。

所有變數

  • dp[201][201][2][201]：DP 陣列
  • MOD：1000000007
  • z：已使用 0 的數量
  • o：已使用 1 的數量
  • b：最後數字（0 或 1）
  • l：尾端連續長度
  • now：目前狀態方案數
  • ans：最終答案
  • k：加總答案時枚舉尾端連續長度

🪜 主要流程步驟

1. 定義 DP 狀態

用 dp[z][o][b][l] 完整描述「數量 + 尾端連續資訊」。

2. 初始化長度為 1 的狀態

dp[1][0][0][1] = 1;
dp[0][1][1][1] = 1;

3. 枚舉所有狀態並轉移

四層迴圈枚舉 z、o、b、l，若 now == 0 就跳過。

4. 當最後是 0 時

  • 若 z < zero 且 l < limit：可再放 0，變成 dp[z + 1][o][0][l + 1]
  • 若 o < one：可改放 1，變成 dp[z][o + 1][1][1]

5. 當最後是 1 時

  • 若 o < one 且 l < limit：可再放 1，變成 dp[z][o + 1][1][l + 1]
  • 若 z < zero：可改放 0，變成 dp[z + 1][o][0][1]

6. 加總最終答案

把所有 dp[zero][one][0][k] 與 dp[zero][one][1][k]（1 <= k <= limit）加總即可。

⏱️ 複雜度

  • 時間複雜度：O(zero * one * limit)
  • 空間複雜度：O(zero * one * limit)

💻 程式碼實作

class Solution {
public:
    int numberOfStableArrays(int zero, int one, int limit) {
        static int dp[201][201][2][201];
        memset(dp, 0, sizeof(dp));
        dp[1][0][0][1] = 1;
        dp[0][1][1][1] = 1;
        const int MOD = 1000000007;

        for (int z = 0; z <= zero; z++) {
            for (int o = 0; o <= one; o++) {
                for (int b = 0; b < 2; b++) {
                    for (int l = 1; l <= limit; l++) {
                        int now = dp[z][o][b][l];
                        if (now == 0) {
                            continue;
                        }
                        if (b == 0) {
                            if (z < zero && l < limit) {
                                dp[z + 1][o][0][l + 1] = (dp[z + 1][o][0][l + 1] + now) % MOD;
                            }
                            if (o < one) {
                                dp[z][o + 1][1][1] = (dp[z][o + 1][1][1] + now) % MOD;
                            }
                        } else {
                            if (o < one && l < limit) {
                                dp[z][o + 1][1][l + 1] = (dp[z][o + 1][1][l + 1] + now) % MOD;
                            }
                            if (z < zero) {
                                dp[z + 1][o][0][1] = (dp[z + 1][o][0][1] + now) % MOD;
                            }
                        }
                    }
                }
            }
        }

        int ans = 0;
        for (int k = 1; k <= limit; k++) {
            ans = (ans + dp[zero][one][0][k]) % MOD;
            ans = (ans + dp[zero][one][1][k]) % MOD;
        }
        return ans;
    }
};
