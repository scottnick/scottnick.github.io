## 網站內容簡介

本網站目前包含以下幾個主要頁面：

### 🧑‍💻 個人頁面
簡單介紹個人背景與目前的學習方向。

### 📘 家教經驗
整理從高中畢業後開始接家教與補習班相關的經驗，
包含教學內容、準備方式與一些個人觀察與反思。

### 📝 修課紀錄
記錄大學期間修習過的課程，作為個人學習歷程的整理。

### 💻 C++ / LeetCode 學習筆記
以 C++ 為主要實作語言，整理 LeetCode 解題筆記與學習紀錄。
目前依照不同題單分類（如 Grind 75、NeetCode 150），
著重於解題思路與觀念整理，而非單純的程式碼答案。
