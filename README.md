# scottnick.github.io

這是我的個人學習紀錄網站，使用 **GitHub Pages** 建立與部署，  
主要用來系統性整理 **大學修課歷程、程式設計學習筆記（C++ / LeetCode）以及教學相關經驗**，  
作為長期累積的學習軌跡與知識整理平台。

🔗 網站連結  
https://scottnick.github.io/

---

## 📌 網站定位與目標

本網站並非單純的作品展示頁，而是：

- 📚 **個人學習歷程的長期紀錄**
- 🧠 **將零散筆記轉化為結構化知識**
- 💻 **以程式與演算法為核心的學習整理站**
- ✍️ **練習將「想法說清楚」而不只是寫出答案**

特別是在 C++ / 演算法筆記中，我更重視：
> **為什麼這樣想、這題在考什麼、可以怎麼優化**

而不只是貼上可 AC 的程式碼。

## 🧭 網站主要內容

### 🧑‍💻 個人頁面（Personal）
- 簡要介紹個人背景、學習方向與目前關注的主題
- 作為整個網站的入口與總覽頁

---

### 📘 家教經驗（Tutoring）
- 紀錄高中畢業後開始從事家教與補習班相關工作的經驗
- 包含：
  - 教學科目與對象
  - 教學準備方式
  - 對學生學習狀況的觀察
  - 教學過程中的反思與調整
- 主要目的是整理自己的教學思維，而非單純列履歷

---

### 📝 修課紀錄（Courses）
- 系統性整理大學期間修習過的課程
- 作為個人學習歷程的備忘與回顧
- 未來可延伸補充：
  - 課程重點
  - 使用到的工具或語言
  - 與其他課程之間的連結

---

### 💻 C++ / LeetCode 學習筆記（Programming Notes）

這是目前網站內容最核心、也持續更新的部分。

- 以 **C++** 為主要實作語言
- 整理 LeetCode 題目與演算法學習紀錄
- 題目依不同題單分類，例如：
  - Grind 75
  - NeetCode 150
  - 其他主題型整理

筆記內容著重於：
- 解題思路與關鍵觀念
- 為何直覺解法會 TLE / WA
- 時間與空間複雜度分析
- 常見陷阱與寫法差異
- 與相關題目的橫向比較

而不是只提供「標準答案」。
