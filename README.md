# scottnick.github.io  
### 個人學習紀錄與程式筆記網站（GitHub Pages）

這是一個由我自行設計、建置並持續維護的 **個人學習紀錄網站**，  
主要用來整合並長期整理：

- 大學修課與學習歷程  
- 家教與教學相關經驗  
- C++ / LeetCode 程式解題與演算法筆記  

本網站不只是作品展示頁，而是一個 **隨著學習同步成長的知識整理系統**。

🔗 網站連結  
https://scottnick.github.io/

---

## 🌐 專案簡介（Project Overview）

此專案為 **靜態個人網站**，透過 GitHub Pages 部署。  
設計目標是將原本分散於筆記平台、零散文件與解題紀錄中的內容，  
整理成一個：

- 結構清楚  
- 可長期維護  
- 容易回顧與擴充  

的個人網站。

---

## 🧭 網站內容與功能導覽

### 🏠 首頁（Home）
- 作為整個網站的入口
- 提供各主要內容區塊的總覽
- 協助使用者快速理解網站用途與結構

---

### 🧑‍💻 個人頁面（Personal）
- 簡要介紹個人背景與目前的學習方向
- 作為整個網站的背景說明頁
- 著重於學習脈絡，而非單純履歷條列

---

### 📘 家教經驗（Tutoring）
- 紀錄高中畢業後開始從事家教與補習班教學的經驗
- 包含教學內容、準備方式與實際觀察
- 目的在於整理教學思維與反思教學過程

---

### 📝 修課紀錄（Courses）
- 系統性整理大學期間修習過的課程
- 作為個人學習歷程的備忘與回顧
- 未來可延伸補充課程重點與學習心得

---

### 💻 C++ / LeetCode 學習筆記（Programming Notes）

此區塊為網站目前最核心、也持續擴充中的內容。

特色包含：

- 以 **C++** 為主要實作語言  
- 整理 LeetCode 題目與演算法學習紀錄  
- 題目依不同題單與主題分類（如 Grind 75、NeetCode 150）  
- 每篇筆記著重於：
  - 解題思路與推導過程
  - 為何直覺解法會 TLE / WA
  - 時間與空間複雜度分析
  - 常見錯誤與寫法比較

目標是建立一套 **可回顧、可延伸的解題知識庫**，  
而非單純收集可通過測資的程式碼。

### 🧠 設計理念
- 不使用後端與資料庫，降低維護成本
- 將重心放在內容結構與可讀性
- 適合作為長期學習紀錄與筆記整理平台
- 可隨需求逐步擴充頁面與內容模組

---

## 📝 內容整理與維護流程

- C++ / 演算法相關筆記最初整理於 **HackMD**
- 隨著內容增多，逐步：
  - 重構筆記結構
  - 統一格式與分類
  - 整合進本網站
- 本網站版本相較原始筆記，更強調：
  - 導覽清楚
  - 主題分類一致
  - 長期可維護性
